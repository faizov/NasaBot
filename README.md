# NasaBot
Телеграм бот NASA.
## API:
https://api.nasa.gov
## Реализовано:
- Получение фото дня. 
- Получение фото Марса, исключаются фотографии маленького размера.
## Планирую реализовать:
- Сделать фото дня раз в день с настройкой от пользователя по времени.
- Получение только цветных фотографий Марса.
